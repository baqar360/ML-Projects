
# Simple Linear Regression - Tip Prediction

## Overview

This project was assigned by my Machine Learning instructor **Sir Zafar Iqbal**.

The goal of this project is to predict the **tip amount** based on the **total bill** using a Simple Linear Regression model.

This project follows a complete machine learning workflow including data exploration, data cleaning, outlier analysis, model training, and evaluation.

---

## Problem Statement

Build a machine learning model that predicts the tip amount using total bill as the input feature.

**Feature:**
- Total Bill

**Target:**
- Tip

---

## Project Workflow

### Exploratory Data Analysis (EDA)
- Understanding dataset structure
- Data visualization
- Finding relationships between variables

### Data Cleaning
- Checking missing values
- Preparing data for training
- Improving data quality

### Outlier Analysis
- Identifying unusual values
- Understanding the effect of outliers on model performance

---

## Machine Learning Model

### Simple Linear Regression

The model learns the relationship between:

**Total Bill → Tip**

and predicts tip values based on the bill amount.

---

## Model Evaluation

Evaluation Metric:

- R² Score

Model Performance:

**R² Score: 52%**
